# Member Verification Site

A lightweight, static membership verification page powered by Google Sheets + Google Apps Script.

## File structure

```
в”ңв”Җв”Җ index.html      вҶҗ main page (do not edit)
```

## Setup

### 1. Google Apps Script (one-time)

1. Open your Google Sheet вҶ’ **Extensions вҶ’ Apps Script**
2. Delete everything and paste the code below
3. Replace `YOUR_SECRET_TOKEN` with the same value in `config.js вҶ’ secretToken`

```javascript
const SECRET_TOKEN = "YOUR_SECRET_TOKEN"; // must match config.js

function doGet(e) {
  // Check token
  const token = e && e.parameter && e.parameter.token;
  if (token !== SECRET_TOKEN) {
    return ContentService
      .createTextOutput(JSON.stringify({ error: "Unauthorized" }))
      .setMimeType(ContentService.MimeType.JSON);
  }

  // Return member data
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const data = sheet.getDataRange().getValues();
  const headers = data[0];
  const rows = data.slice(1).map(row => {
    const obj = {};
    headers.forEach((h, i) => { obj[h] = row[i]; });
    return obj;
  });

  return ContentService
    .createTextOutput(JSON.stringify(rows))
    .setMimeType(ContentService.MimeType.JSON);
}
```

4. Click **Save** вҶ’ **Deploy вҶ’ New deployment**
5. Gear icon вҡҷ вҶ’ **Web app**
6. Set **Who has access** вҶ’ **Anyone**
7. Click **Deploy** вҶ’ copy the **Web app URL**
8. Paste the URL into `config.js вҶ’ scriptUrl`

> вҡ пёҸ After any change to the Apps Script code, you must **Deploy вҶ’ New deployment** (not "Manage deployments") to apply it.

### 2. config.js

```js
const CONFIG = {
  orgName:     "Your Organization Name",
  scriptUrl:   "https://script.google.com/macros/s/YOUR_ID/exec",
  secretToken: "YOUR_SECRET_TOKEN",   // same as in Apps Script

  col_memberId:   "ID",
  col_name:       "Nama",
  col_status:     "status",
  col_expiry:     "Masa_berlaku",
  col_memberType: "",
};
```

## Security notes

- The token stops random people from querying your Apps Script URL directly
- The token is visible in page source вҖ” this is a deterrent, not a hard lock
- Do not store sensitive data (passwords, addresses, phone numbers) in the sheet
- The sheet is **read-only** вҖ” nothing can be written through this site
